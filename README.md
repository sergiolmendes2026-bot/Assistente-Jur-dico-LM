## ⚖️ Assistente Jurídico com RAG

Este projeto incorpora um pipeline de **RAG (Retrieval-Augmented Generation)**, utilizando modelos de *embeddings* e banco vetorial.  
É um excelente exemplo para quem deseja compreender o que realmente é **Inteligência Artificial** e como funciona uma aplicação de **IA Generativa**, apresentada passo a passo.

### 🔎 Contexto
Em muitas empresas, pode ser necessário consultar documentos internos. 
É exatamente nesse cenário que o RAG se torna útil: ele funciona como uma infraestrutura de backend, semelhante a um banco de dados, rodando nos servidores da organização.  
Ao configurar a **API Key**, o sistema ativa automaticamente a caixa de entrada para o usuário digitar sua mensagem e também o campo para upload de documentos em PDF.

### 🛠️ Configuração
Nesta aplicação, configurei o RAG para trabalhar exclusivamente com documentos PDF.  
O objetivo foi tornar o assistente bem específico: o usuário deve inserir mensagens relacionadas apenas ao PDF que foi carregado.
